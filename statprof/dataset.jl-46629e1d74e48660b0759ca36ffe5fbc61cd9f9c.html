101 """
102 NCDataset(filename::AbstractString, mode = "r";
103 format::Symbol = :netcdf4,
104 share::Bool = false,
105 diskless::Bool = false,
106 persist::Bool = false,
107 memory::Union{Vector{UInt8},Nothing} = nothing,
108 attrib = [])
109
110 Load, create, or even overwrite a NetCDF file at `filename`, depending on `mode`
111
112 * `"r"` (default) : open an existing netCDF file or OPeNDAP URL
113 in read-only mode.
114 * `"c"` : create a new NetCDF file at `filename` (an existing file with the same
115 name will be overwritten).
116 * `"a"` : open `filename` into append mode (i.e. existing data in the netCDF
117 file is not overwritten and a variable can be added).
118
119
120 If `share` is true, the `NC_SHARE` flag is set allowing to have multiple
121 processes to read the file and one writer process (netcdf classic files only).
122 Likewise setting `diskless` or `persist` to `true` will enable the flags
123 `NC_DISKLESS` or `NC_PERSIST` flag.
124 More information is available in the [NetCDF C-API](https://www.unidata.ucar.edu/software/netcdf/docs/).
125
126 Notice that this does not close the dataset, use `close` on the
127 result (or see below the `do`-block).
128
129 The optional parameter `attrib` is an iterable of attribute name and attribute
130 value pairs, for example a `Dict`, `DataStructures.OrderedDict` or simply a
131 vector of pairs (see example below).
132
133 # Supported `format` values:
134
135 * `:netcdf4` (default): HDF5-based NetCDF format.
136 * `:netcdf4_classic`: Only netCDF 3 compatible API features will be used.
137 * `:netcdf3_classic`: classic netCDF format supporting only files smaller than 2GB.
138 * `:netcdf3_64bit_offset`: improved netCDF format supporting files larger than 2GB.
139 * `:netcdf5_64bit_data`: improved netCDF format supporting 64-bit integer data types.
140
141
142 Files can also be open and automatically closed with a `do` block.
143
144 ```julia
145 NCDataset("file.nc") do ds
146 data = ds["temperature"][:,:]
147 end
148 ```
149
150 Here is an attribute example:
151 ```julia
152 using DataStructures
153 NCDataset("file.nc", "c", attrib = OrderedDict("title" => "my first netCDF file")) do ds
154 defVar(ds,"temp",[10.,20.,30.],("time",))
155 end;
156 ```
157
158 The NetCDF dataset can also be a `memory` as a vector of bytes. A non-empty string
159 a `filename` is still required, for example:
160
161 ```julia
162 using NCDataset, HTTP
163 resp = HTTP.get("https://www.unidata.ucar.edu/software/netcdf/examples/ECMWF_ERA-40_subset.nc")
164 ds = NCDataset("some_string","r",memory = resp.body)
165 total_precipitation = ds["tp"][:,:,:]
166 close(ds)
167 ```
