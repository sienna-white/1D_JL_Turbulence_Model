33 """
34 v = cfvariable(ds::NCDataset,varname::SymbolOrString; <attrib> = <value>)
35
36 Return the variable `varname` in the dataset `ds` as a
37 `NCDataset.CFVariable`. The keyword argument `<attrib>` are
38 the attributes (`fillvalue`, `missing_value`, `scale_factor`, `add_offset`,
39 `units` and `calendar`) relevant to the CF conventions.
40 By specifing the value of these attributes, the one can override the value
41 specified in the data set. If the attribute is set to `nothing`, then
42 the attribute is not loaded and the corresponding transformation is ignored.
43 This function is similar to `ds[varname]` with the additional flexibility that
44 some variable attributes can be overridden.
45
46
47 Example:
48
49 ```julia
50 NCDataset("foo.nc","c") do ds
51 defVar(ds,"data",[10., 11., 12., 13.], ("time",), attrib = Dict(
52 "add_offset" => 10.,
53 "scale_factor" => 0.2))
54 end
55
56 # The stored (packed) valued are [0., 5., 10., 15.]
57 # since 0.2 .* [0., 5., 10., 15.] .+ 10 is [10., 11., 12., 13.]
58
59 ds = NCDataset("foo.nc");
60
61 @show ds["data"].var[:]
62 # returns [0., 5., 10., 15.]
63
64 @show cfvariable(ds,"data")[:]
65 # returns [10., 11., 12., 13.]
66
67 # neither add_offset nor scale_factor are applied
68 @show cfvariable(ds,"data", add_offset = nothing, scale_factor = nothing)[:]
69 # returns [0, 5, 10, 15]
70
71 # add_offset is applied but not scale_factor
72 @show cfvariable(ds,"data", scale_factor = nothing)[:]
73 # returns [10, 15, 20, 25]
74
75 # 0 is declared as the fill value (add_offset and scale_factor are applied as usual)
76 @show cfvariable(ds,"data", fillvalue = 0)[:]
77 # returns [missing, 11., 12., 13.]
78
79 # Use the time units: days since 2000-01-01
80 @show cfvariable(ds,"data", units = "days since 2000-01-01")[:]
