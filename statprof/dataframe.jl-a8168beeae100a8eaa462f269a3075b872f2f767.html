2 DataFrame <: AbstractDataFrame
3
4 An `AbstractDataFrame` that stores a set of named columns.
5
6 The columns are normally `AbstractVector`s stored in memory,
7 particularly a `Vector`, `PooledVector` or `CategoricalVector`.
8

10 ```julia

23 DataFrame(::GroupedDataFrame; copycols::Bool=true, keepkeys::Bool=true)
24 ```
25
26 # Keyword arguments
27
28 - `copycols` : whether vectors passed as columns should be copied; by default set
29 to `true` and the vectors are copied; if set to `false` then the constructor
30 will still copy the passed columns if it is not possible to construct a
31 `DataFrame` without materializing new columns. Note the `copycols=nothing`
32 default in the Tables.jl compatible constructor; it is provided as certain
33 input table types may have already made a copy of columns or the columns may
34 otherwise be immutable, in which case columns are not copied by default.
35 To force a copy in such cases, or to get mutable columns from an immutable
36 input table (like `Arrow.Table`), pass `copycols=true` explicitly.
37 - `makeunique` : if `false` (the default), an error will be raised
38
39 (note that not all constructors support these keyword arguments)
40
41 # Details on behavior of different constructors
42
43 It is allowed to pass a vector of `Pair`s, a list of `Pair`s as positional
44 arguments, or a list of keyword arguments. In this case each pair is considered
45 to represent a column name to column value mapping and column name must be a
46 `Symbol` or string. Alternatively a dictionary can be passed to the constructor
47 in which case its entries are considered to define the column name and column
48 value pairs. If the dictionary is a `Dict` then column names will be sorted in
49 the returned `DataFrame`.
50
51 In all the constructors described above column value can be a vector which is
52 consumed as is or an object of any other type (except `AbstractArray`). In the
53 latter case the passed value is automatically repeated to fill a new vector of
54 the appropriate length. As a particular rule values stored in a `Ref` or a
55 `0`-dimensional `AbstractArray` are unwrapped and treated in the same way.
56
57 It is also allowed to pass a vector of vectors or a matrix as as the first
58 argument. In this case the second argument must be
59 a vector of `Symbol`s or strings specifying column names, or the symbol `:auto`
60 to generate column names `x1`, `x2`, ... automatically. Note that in this case
61 if the first argument is a matrix and `copycols=false` the columns of the created
62 `DataFrame` will be views of columns the source matrix.
63
64 If a single positional argument is passed to a `DataFrame` constructor then it
65 is assumed to be of type that implements the
66 [Tables.jl](https://github.com/JuliaData/Tables.jl) interface using which the
67 returned `DataFrame` is materialized.
68
69 If two positional arguments are passed, where the second argument is an
70 `AbstractVector`, then the first argument is taken to be a table as described in
71 the previous paragraph, and columns names of the resulting data frame
72 are taken from the vector passed as the second positional argument.
73
74 Finally it is allowed to construct a `DataFrame` from a `DataFrameRow` or a
75 `GroupedDataFrame`. In the latter case the `keepkeys` keyword argument specifies
76 whether the resulting `DataFrame` should contain the grouping columns of the
77 passed `GroupedDataFrame` and the order of rows in the result follows the order
78 of groups in the `GroupedDataFrame` passed.
79
80 # Notes
81
82 The `DataFrame` constructor by default copies all columns vectors passed to it.
83 Pass the `copycols=false` keyword argument (where supported) to reuse vectors without
84 copying them.
85
86 By default an error will be raised if duplicates in column names are found. Pass
87 `makeunique=true` keyword argument (where supported) to accept duplicate names,
88 in which case they will be suffixed with `_i` (`i` starting at 1 for the first
89 duplicate).
90
91 If an `AbstractRange` is passed to a `DataFrame` constructor as a column it is
92 always collected to a `Vector` (even if `copycols=false`). As a general rule
93 `AbstractRange` values are always materialized to a `Vector` by all functions in
94 DataFrames.jl before being stored in a `DataFrame`.
95
96 `DataFrame` can store only columns that use 1-based indexing. Attempting
97 to store a vector using non-standard indexing raises an error.
98
99 The `DataFrame` type is designed to allow column types to vary and to be
100 dynamically changed also after it is constructed. Therefore `DataFrame`s are not
101 type stable. For performance-critical code that requires type-stability either
102 use the functionality provided by `select`/`transform`/`combine` functions, use
103 `Tables.columntable` and `Tables.namedtupleiterator` functions, use barrier
104 functions, or provide type assertions to the variables that hold columns
105 extracted from a `DataFrame`.
106
107 Metadata: this function preserves all table and column-level metadata.
108 As a special case if a `GroupedDataFrame` is passed then
109 only `:note`-style metadata from parent of the `GroupedDataFrame` is preserved.
110

193 function DataFrame(columns::Union{Vector{Any}, Vector{AbstractVector}},
194 colindex::Index; copycols::Bool=true)
195 if length(columns) == length(colindex) == 0
196 return new(AbstractVector[], Index(), nothing, nothing, true)
197 elseif length(columns) != length(colindex)
198 throw(DimensionMismatch("Number of columns ($(length(columns))) and number of " *
199 "column names ($(length(colindex))) are not equal"))
200 end
201
202 len = -1
203 firstvec = -1
204 for (i, col) in enumerate(columns)
205 if col isa AbstractVector
206 if len == -1
207 len = length(col)
208 firstvec = i
209 elseif len != length(col)
210 n1 = _names(colindex)[firstvec]
211 n2 = _names(colindex)[i]
212 throw(DimensionMismatch("column :$n1 has length $len and column " *
213 ":$n2 has length $(length(col))"))
214 end
215 end
216 end
217 len == -1 && (len = 1) # we got no vectors so make one row of scalars
218
219 # we write into columns as we know that it is guaranteed
220 # that it was freshly allocated in the outer constructor
221 if copycols && len >= 100_000 && length(columns) > 1 && Threads.nthreads() > 1
222 @sync for i in eachindex(columns)
223 @spawn columns[i] = _preprocess_column(columns[i], len, copycols)
224 end
225 else
226 for i in eachindex(columns)
227 columns[i] = _preprocess_column(columns[i], len, copycols)
228 end
229 end
230
231 for (i, col) in enumerate(columns)
232 firstindex(col) != 1 && _onebased_check_error(i, col)
233 end
234
235 return new(convert(Vector{AbstractVector}, columns), colindex, nothing, nothing, true)
236 end

239 function _preprocess_column(col::Any, len::Integer, copycols::Bool)
240 if col isa AbstractRange
241 return collect(col)
242 elseif col isa AbstractVector
243 return copycols ? copy(col) : col
244 elseif col isa Union{AbstractArray{<:Any, 0}, Ref}
245 x = col[]
246 return fill!(Tables.allocatecolumn(typeof(x), len), x)
247 elseif col isa AbstractArray
248 throw(ArgumentError("adding AbstractArray other than AbstractVector " *
249 "as a column of a data frame is not allowed"))
250 else
251 return fill!(Tables.allocatecolumn(typeof(col), len), col)
252 end

473 corrupt_msg(df::DataFrame, i::Integer) =
474 "Data frame is corrupt: length of column " *
475 ":$(_names(df)[i]) ($(length(df[!, i]))) " *
476 "does not match length of column 1 ($(length(df[!, 1]))). " *
477 "The column vector has likely been resized unintentionally " *
478 "(either directly or because it is shared with another data frame)."

480 function _check_consistency(df::DataFrame)
481 cols, idx = _columns(df), index(df)
482
483 for (i, col) in enumerate(cols)
484 firstindex(col) != 1 && _onebased_check_error(i, col)
485 end
486
487 ncols = length(cols)
488 @assert length(idx.names) == length(idx.lookup) == ncols
489 ncols == 0 && return nothing
490 nrows = length(cols[1])
491 for i in 2:length(cols)
492 @assert length(cols[i]) == nrows corrupt_msg(df, i)
493 end

864 function Base.deleteat!(df::DataFrame, inds::AbstractVector)
865 if isempty(inds)
866 _drop_all_nonnote_metadata!(df)
867 return df
868 elseif size(df, 2) == 0
869 throw(BoundsError(df, (inds, :)))
870 end
871
872 if Bool <: eltype(inds) && any(x -> x isa Bool, inds)
873 throw(ArgumentError("invalid index of type Bool"))
874 end
875
876 if !(eltype(inds) <: Integer || all(x -> x isa Integer, inds))
877 throw(ArgumentError("unsupported index $inds"))
878 end
879
880 # workaround https://github.com/JuliaLang/julia/pull/41646
881 if VERSION <= v"1.6.2" && inds isa UnitRange{<:Integer}
882 inds = collect(inds)
883 end
884
885 if !issorted(inds, lt=<=)
886 throw(ArgumentError("Indices passed to deleteat! must be unique and sorted"))
887 end

890 end
891
892 function Base.deleteat!(df::DataFrame, inds::AbstractVector{Bool})
893 if length(inds) != size(df, 1)
894 throw(BoundsError(df, (inds, :)))
895 end
896 # workaround https://github.com/JuliaLang/julia/pull/41646
897 if VERSION <= v"1.6.2" && drop isa UnitRange{<:Integer}
898 inds = collect(inds)
899 end
900 return _deleteat!_helper(df, inds)
901 end
902
903 Base.deleteat!(df::DataFrame, inds::Not) =
904 _deleteat!_helper(df, axes(df, 1)[inds])
905
906 function _deleteat!_helper(df::DataFrame, drop)
907 cols = _columns(df)
908 if isempty(cols)
909 _drop_all_nonnote_metadata!(df)
910 return df
911 end
912
913 if any(c -> c === drop || Base.mightalias(c, drop), cols)
914 drop = copy(drop)
915 end
916
917 n = nrow(df)
918 col1 = cols[1]
919 deleteat!(col1, drop)
920 newn = length(col1)
921 @assert newn <= n
922 # the 0.06 threshold is heuristic; based on tests
923 # the assignment makes the code type-unstable but it is a small union
924 # so the overhead should be small
925 if drop isa AbstractVector{Bool} && newn < 0.06 * n
926 drop = findall(drop)
927 end
928 for i in 2:length(cols)
929 col = cols[i]
930 # this check is done to handle column aliases
931 if length(col) == n
932 deleteat!(col, drop)
933 end
934 end
935
936 for i in 1:length(cols)
937 # this should never happen, but we add it for safety
938 @assert length(cols[i]) == newn corrupt_msg(df, i)
939 end
940
941 _drop_all_nonnote_metadata!(df)
942 return df
943 end

1359 Update a data frame `df` in-place by repeating its rows. `inner` specifies how many

1361 of rows is repeated. Columns of `df` are freshly allocated.

1383 2 │ 1 3
1384 3 │ 2 4
1385 4 │ 2 4
1386 5 │ 1 3
1387 6 │ 1 3
1388 7 │ 2 4
1389 8 │ 2 4
1390 9 │ 1 3
1391 10 │ 1 3
1392 11 │ 2 4
1393 12 │ 2 4
1394 ```

1396 function repeat!(df::DataFrame; inner::Integer=1, outer::Integer=1)
1397 inner < 0 && throw(ArgumentError("inner keyword argument must be non-negative"))
1398 outer < 0 && throw(ArgumentError("outer keyword argument must be non-negative"))
1399 cols = _columns(df)
1400 for (i, col) in enumerate(cols)
1401 col_new = repeat(col, inner=Int(inner), outer=Int(outer))
1402 firstindex(col_new) != 1 && _onebased_check_error(i, col_new)
1403 cols[i] = col_new
1404 end
1405 _drop_all_nonnote_metadata!(df)
1406 return df
1407 end

1480 """
1481 allcombinations(DataFrame, pairs::Pair...)
1482 allcombinations(DataFrame; kwargs...)
1483
1484 Create a `DataFrame` from all combinations of values in passed arguments.
1485 The first passed values vary fastest.
1486
1487 Arguments associating a column name with values to expand can be specified
1488 either as `Pair`s passed as positional arguments, or as keyword arguments.
1489 Column names must be `Symbol`s or strings and must be unique.
1490
1491 Column value can be a vector which is consumed as is or an object of any other
1492 type (except `AbstractArray`). In the latter case the passed value is treated
1493 as having length one for expansion. As a particular rule values stored in a `Ref`
1494 or a `0`-dimensional `AbstractArray` are unwrapped and treated as having length one.
1495
1496 See also: [`crossjoin`](@ref) can be used to get the cartesian product
1497 of rows from passed data frames.
1498
1499 # Examples
1500
1501 ```jldoctest
1502 julia> allcombinations(DataFrame, a=1:2, b='a':'c')
1503 6×2 DataFrame
1504 Row │ a b
1505 │ Int64 Char
1506 ─────┼─────────────
1507 1 │ 1 a
1508 2 │ 2 a
1509 3 │ 1 b
1510 4 │ 2 b
1511 5 │ 1 c
1512 6 │ 2 c
1513
1514 julia> allcombinations(DataFrame, "a" => 1:2, "b" => 'a':'c', "c" => "const")
1515 6×3 DataFrame
1516 Row │ a b c
1517 │ Int64 Char String
1518 ─────┼─────────────────────
1519 1 │ 1 a const
1520 2 │ 2 a const
1521 3 │ 1 b const
1522 4 │ 2 b const
1523 5 │ 1 c const
1524 6 │ 2 c const
1525 ```
1526 """
1527 function allcombinations(::Type{DataFrame}, pairs::Pair{Symbol, <:Any}...)
1528 colnames = first.(pairs)
1529 if !allunique(colnames)
1530 throw(ArgumentError("All column names passed to allcombinations must be unique"))
1531 end
1532 colvalues = map(pairs) do p
1533 v = last(p)
1534 if v isa AbstractVector
1535 return v
1536 elseif v isa Union{AbstractArray{<:Any, 0}, Ref}
1537 x = v[]
1538 return fill!(Tables.allocatecolumn(typeof(x), 1), x)
1539 elseif v isa AbstractArray
1540 throw(ArgumentError("adding AbstractArray other than AbstractVector " *
1541 "as a column of a data frame is not allowed"))
1542 else
1543 return fill!(Tables.allocatecolumn(typeof(v), 1), v)
1544 end
1545 end
1546 @assert length(colvalues) == length(colnames)
1547 @assert all(x -> x isa AbstractVector, colvalues)
1548
1549 target_rows = Int(prod(x -> BigInt(length(x)), colvalues))
1550 out_df = DataFrame()
1551 inner = 1
1552 for (val, cname) in zip(colvalues, colnames)
1553 len = length(val)
1554 last_inner = inner
1555 inner *= len
1556 outer, remv = inner == 0 ? (0, 0) : divrem(target_rows, inner)
1557 @assert iszero(remv)
1558 out_df[!, cname] = repeat(val, inner=last_inner, outer=outer)
1559 end
1560 @assert inner == target_rows
1561 @assert size(out_df) == (target_rows, length(colnames))
1562 return out_df
1563 end
