141 """
142 ord(lt, by, rev::Union{Bool, Nothing}, order::Ordering=Forward)
143
144 Construct an [`Ordering`](@ref) object from the same arguments used by
145 [`sort!`](@ref).
146 Elements are first transformed by the function `by` (which may be
147 [`identity`](@ref)) and are then compared according to either the function `lt`
148 or an existing ordering `order`. `lt` should be [`isless`](@ref) or a function
149 that obeys the same rules as the `lt` parameter of [`sort!`](@ref). Finally,
150 the resulting order is reversed if `rev=true`.
151
152 Passing an `lt` other than `isless` along with an `order` other than
153 [`Base.Order.Forward`](@ref) or [`Base.Order.Reverse`](@ref) is not permitted,
154 otherwise all options are independent and can be used together in all possible
155 combinations.
156 """
157 ord(lt, by, rev::Nothing, order::Ordering=Forward) = _ord(lt, by, order)

159 function ord(lt, by, rev::Bool, order::Ordering=Forward)
160 o = _ord(lt, by, order)
161 return rev ? ReverseOrdering(o) : o
162 end
