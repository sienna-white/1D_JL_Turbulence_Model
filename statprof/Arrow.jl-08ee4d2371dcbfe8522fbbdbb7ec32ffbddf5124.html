17 """
18 Arrow.jl
19
20 A pure Julia implementation of the [apache arrow](https://arrow.apache.org/) memory format specification.
21
22 This implementation supports the 1.0 version of the specification, including support for:
23 * All primitive data types
24 * All nested data types
25 * Dictionary encodings, nested dictionary encodings, and messages
26 * Extension types
27 * Streaming, file, record batch, and replacement and isdelta dictionary messages
28 * Buffer compression/decompression via the standard LZ4 frame and Zstd formats
29
30 It currently doesn't include support for:
31 * Tensors or sparse tensors
32 * Flight RPC
33 * C data interface
34
35 Third-party data formats:
36 * csv and parquet support via the existing [CSV.jl](https://github.com/JuliaData/CSV.jl) and [Parquet.jl](https://github.com/JuliaIO/Parquet.jl) packages
37 * Other [Tables.jl](https://github.com/JuliaData/Tables.jl)-compatible packages automatically supported ([DataFrames.jl](https://github.com/JuliaData/DataFrames.jl), [JSONTables.jl](https://github.com/JuliaData/JSONTables.jl), [JuliaDB.jl](https://github.com/JuliaData/JuliaDB.jl), [SQLite.jl](https://github.com/JuliaDatabases/SQLite.jl), [MySQL.jl](https://github.com/JuliaDatabases/MySQL.jl), [JDBC.jl](https://github.com/JuliaDatabases/JDBC.jl), [ODBC.jl](https://github.com/JuliaDatabases/ODBC.jl), [XLSX.jl](https://github.com/felipenoris/XLSX.jl), etc.)
38 * No current Julia packages support ORC or Avro data formats
39
40 See docs for official Arrow.jl API with the [User Manual](@ref) and reference docs for [`Arrow.Table`](@ref), [`Arrow.write`](@ref), and [`Arrow.Stream`](@ref).
41 """

112 function access_threaded(f, v::Vector)
113 tid = Threads.threadid()
114 0 < tid <= length(v) || _length_assert()
115 if @inbounds isassigned(v, tid)
116 @inbounds x = v[tid]
117 else
118 x = f()
119 @inbounds v[tid] = x
120 end
121 return x
122 end
123 @noinline _length_assert() = @assert false "0 < tid <= v"
