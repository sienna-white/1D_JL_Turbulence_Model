186 """
187 varbyattrib(ds, attname = attval)
188
189 Returns a list of variable(s) which has the attribute `attname` matching the value `attval`
190 in the dataset `ds`.
191 The list is empty if the none of the variables has the match.
192 The output is a list of `CFVariable`s.
193
194 # Examples
195
196 Load all the data of the first variable with standard name "longitude" from the
197 NetCDF file `results.nc`.
198
199 ```julia-repl
200 julia> ds = NCDataset("results.nc", "r");
201 julia> data = varbyattrib(ds, standard_name = "longitude")[1][:]
202 ```
203
204 """
205 function varbyattrib(ds::Union{AbstractDataset,AbstractVariable}; kwargs...)
206 # Start with an empty list of variables
207 varlist = []
208
209 # Loop on the variables
210 for v in keys(ds)
211 var = ds[v]
212
213 matchall = true
214
215 for (attsym,attval) in kwargs
216 attname = String(attsym)
217
218 # Check if the variable has the desired attribute
219 if attname in attribnames(var)
220 # Check if the attribute value is the selected one
221 if attrib(var,attname) != attval
222 matchall = false
223 break
224 end
225 else
226 matchall = false
227 break
228 end
229 end
230
231 if matchall
232 push!(varlist, var)
233 end
234 end
235
236 return varlist
237 end
